# konfhub-frontend-task

This project is built for the **KonfHub Frontend Engineer Evaluation** using **React, Vite, and Tailwind CSS**.  
The task is to fetch event data from the KonfHub public API and display it in a clean, modular, and user-friendly UI.

---------------------------------------------------------------------------------------------------

## Tech Stack Used

**Frontend**
- **React 18**
- **Vite 5 (Bundler + Dev Server)**
- **Tailwind CSS 3.4 (Utility-first styling)**
- **JavaScript (ES6+)**
- **Fetch API for API calls**

 **Tools & Configuration**
- **Node.js (v18+ recommended)**
- **NPM (Package Manager)**
- **PostCSS & Autoprefixer** (for Tailwind processing)
- **Git & GitHub** (Version control)
